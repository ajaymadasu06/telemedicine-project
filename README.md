# Telemedicine for Rural Healthcare

A real-time telemedicine web application designed to support rural ambulance services with instant doctor guidance during emergencies.

## Features
- Patient details management
- Doctor and paramedic interface
- Emergency case tracking
- Real-time healthcare support
- Rural healthcare assistance

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Project Objective
The project aims to reduce emergency response time and improve communication between doctors and paramedics in rural areas.

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run:
   python app.py
